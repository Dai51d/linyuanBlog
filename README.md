## 简介
这是一个基于[vuepress](https://vuepress.vuejs.org/)的博客网站，使用的主题是[vuepress-theme-reco](https://vuepress-theme-reco.recoluan.com/)。

## 目录结构
```
azusaWebBlog
- blogs
  + .vuepress
    + public
      ...
    + config.js
  + frontEnd
    ...
  + leetcode
    ...
  + other
    ...
- gitignore
- deploy.sh
- package.json
- README.md
```

## 安装依赖
```
npm install
```

## 运行
```
npm run dev
```
